# Global Logistics Performance Dashboard

## Project Overview
This project analyzes over 181,000 global shipment records to identify supply chain inefficiencies, monitor delivery deadlines, and isolate revenue risks across international markets.

## Tools Used
* **MySQL:** Used for data cleaning and extraction
* **Power BI:** Relational data modeling (Star Schema), advanced DAX measures, and interactive dashboard design.

## Key Insights

- **Revenue Risk Identified:** Detected a global **57.28% late delivery rate**, placing **$18.89M** in revenue at operational risk.  

- **Data Integrity Discovery:** Identified a shipping data anomaly where *First Class* orders showed an artificial **100% failure rate** across **27.8K shipments**, indicating potential upstream data quality issues.  

- **Operational Bottlenecks:** Analyzed transit performance to identify severe delivery delays in high-risk regions, led by **Ecuador (69.39%)** and **Niger (66.67%)**.  

- **Fulfillment Severity:** Developed operational KPIs showing monthly late deliveries consistently ranging between **7,100 and 10,200 orders**, with an average delay of **1.62 days** beyond promised SLAs.
